# Milestone v1.0: MVP

**Status:** SHIPPED 2026-03-12

## Overview

Yogi Assistant is an AI-powered developer portfolio presented as an interactive terminal console. The roadmap progressed from data foundation through static UI, terminal implementation, terminal polish, AI integration, production readiness, and AI caching/smart routing.

## Milestone Summary

**Key Decisions:**
- React Three Fiber + Rapier for interactive 3D badge (replaced glassmorphism card)
- Vercel AI SDK v6 (not raw Anthropic SDK) for streaming
- Claude Haiku 4.5 for AI responses (cost-efficient)
- Three-tier response system: static -> cache -> API call
- Tailwind v4 CSS-based config, OKLCH color space
- Intent-based routing with keyword matching (zero AI calls for classification)

**Issues Resolved:**
- WebGL Context Lost on Canvas double-mount (fixed with reactStrictMode: false)
- AI SDK v6 breaking changes (async convertToModelMessages, LanguageModelV3 stream protocol)
- Intent router false positives (reordered rules, narrowed keywords)
- Portfolio data mismatch with resume (synced all data)

**Technical Debt Incurred:**
- Tab completion UX on mobile needs design decision
- No real profile photo (placeholder avatar)
