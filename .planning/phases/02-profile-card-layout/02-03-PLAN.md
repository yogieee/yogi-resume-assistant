<task type="auto">
  <name>Task 1: Create ProfileCard component with glassmorphism, data, and animations</name>
  <files>src/components/profile-card.tsx</files>
  <action>
Create the profile card as a client component (`"use client"`) since it uses motion animations and state.

**Imports:**
- `motion` from `"motion/react"` (NOT framer-motion)
- `useState` from `"react"`
- `MapPin`, `Download` from `"lucide-react"`
- `buttonVariants` from `"@/components/ui/button"` (use buttonVariants on `<a>` tag, NOT Button with asChild -- base-ui Button may not support asChild, see Research Open Question 1)
- `SocialLinks` from `"@/components/social-links"`
- `portfolio` from `"@/data/portfolio"`

**Data binding:**
- Destructure `const { about, contact } = portfolio` at module level
- All displayed text comes from these data objects, never hardcoded

**Component structure (inside a `motion.div`):**

1. **Avatar placeholder:** A `size-24 rounded-full` div with `bg-console-elevated border-2 border-console-border` centered, containing bold "YG" initials in `text-glow-cyan text-2xl`. Use `flex items-center justify-center`.

2. **Identity section:** Centered text with:
   - `<h1>` with `text-xl font-bold text-console-text-bright` showing `about.name`
   - `<p>` with `text-sm text-glow-cyan` showing `about.role`
   - `<p>` with `text-xs text-console-text-dim` showing `about.tagline`

3. **Location:** Flex row with `MapPin` icon (`size-3.5`) and `about.location`, styled `text-sm text-console-text-dim`

4. **Social links:** `<SocialLinks contacts={contact} />`

5. **Resume download:** An `<a>` tag with `href="/resume.pdf" download` styled using `buttonVariants({ variant: "outline" })` plus `w-full gap-2 justify-center`. Include `<Download className="size-4" />` icon and text "Download Resume".

**Glassmorphism card wrapper (inside the motion.div):**
```
className="relative rounded-2xl p-6 space-y-5
  bg-console-surface/80 backdrop-blur-xl
  border border-console-border/50
  shadow-lg shadow-black/20 ring-1 ring-white/5"
```

**Outer motion.div wrapper:**
- `className="w-full max-w-sm mx-auto lg:max-w-none"`

**Animation: entrance + floating transition pattern** (from Research Pattern 5):
- Use `useState` to track `hasEntered` (defaults to `false`)
- `initial={{ opacity: 0, y: 30, scale: 0.95 }}`
- `animate`:
  - When `!hasEntered`: `{ opacity: 1, y: 0, scale: 1 }` (entrance)
  - When `hasEntered`: `{ opacity: 1, y: [0, -8, 0], scale: 1 }` (floating loop)
- `transition`:
  - When `!hasEntered`: `{ duration: 0.6, ease: "easeOut" }`
  - When `hasEntered`: `{ y: { duration: 4, ease: "easeInOut", repeat: Infinity } }`
- `onAnimationComplete`: set `hasEntered(true)` when `!hasEntered`

This gives a smooth entrance (fade in + slide up + scale) that transitions into a continuous subtle float.
