**However**, for Tier 1 (static responses), the cleanest approach is to bypass streamText entirely and use `createUIMessageStream` or construct a compatible response directly. The research notes that `simulateReadableStream` works within middleware's `wrapStream`, but for Tier 1 we want to skip the model entirely.

**Recommended approach for Tier 1:** Use `streamText` with the wrapped model, but have the middleware handle ALL three tiers. The intent classification happens inside the middleware's wrapStream, which can short-circuit with a static response before calling doStream(). This is the cleanest approach per the research's Open Question 1 recommendation.

**REVISED approach (simpler, more reliable):** Actually, handle Tier 1 in the route handler BEFORE streamText. Use the AI SDK's `createUIMessageStreamResponse` helper or manually construct a DataStream-compatible response for static content. Here's why: the middleware doesn't have access to the original user message text easily (it gets transformed params), but the route handler has the raw messages.

**FINAL approach:** Handle Tier 1 in the route handler. For known topics, create a response using `streamText` with the wrapped model -- the caching middleware will see this and can handle it. BUT this still calls the model for a known topic on first hit.

import { streamText, UIMessage, convertToModelMessages, wrapLanguageModel } from "ai";
import { anthropic } from "@ai-sdk/anthropic";
import { getEnv } from "@/lib/env";
import { rateLimit } from "@/lib/ai/rate-limit";
import { classifyIntent } from "@/lib/ai/intent-router";
import { getStaticResponse } from "@/lib/ai/static-responses";
import { buildMinimalContext } from "@/lib/ai/topic-context";
import { cachingMiddleware } from "@/lib/ai/caching-middleware";

// Create wrapped model with caching middleware (module-level, created once)
const cachedModel = wrapLanguageModel({
  model: anthropic("claude-haiku-4-5-20251001"),
  middleware: cachingMiddleware,
});

export async function POST(req: Request) {
  // Rate limit (unchanged)
  const ip = req.headers.get("x-forwarded-for") ?? req.headers.get("x-real-ip") ?? "unknown";
  const { success } = rateLimit(ip);
  if (!success) {
    return Response.json(
      { error: "Too many requests. Please try again in a moment." },
      { status: 429 },
    );
  }

  getEnv();

  try {
    const { messages }: { messages: UIMessage[] } = await req.json();

    // Extract last user message
    const lastMessage = messages.filter(m => m.role === "user").pop();
    const lastText = lastMessage?.parts
      ?.filter((p): p is { type: "text"; text: string } => p.type === "text")
      .map(p => p.text)
      .join(" ") ?? "";

    // Tier 1: Known topic -> static response (no AI call)
    const intent = classifyIntent(lastText);

    if (intent !== "novel") {
      const staticText = getStaticResponse(intent);
      // Return static response through AI SDK streaming protocol
      // Try createUIMessageStream approach first
      // If that doesn't work, use streamText with a system-only approach
      // See verification step -- test which approach works
    }

    // Tier 2 & 3: Cache middleware handles cache hit/miss
    const result = streamText({
      model: cachedModel,
      system: buildMinimalContext(intent !== "novel" ? intent : null),
      messages: await convertToModelMessages(messages.slice(-20)),
      temperature: 0,
      maxOutputTokens: 1024,
    });

    return result.toUIMessageStreamResponse();
  } catch (error) {
    console.error("Chat API error:", error);
    return Response.json(
      { error: "Something went wrong. Please try again." },
      { status: 500 },
    );
  }
}
```

import type { LanguageModelV3StreamPart } from "@ai-sdk/provider";

export function textToStreamChunks(text: string): LanguageModelV3StreamPart[] {
  const words = text.split(/(\s+)/);
  const chunks: LanguageModelV3StreamPart[] = words.map((word) => ({
    type: "text-delta" as const,
    textDelta: word,
  }));
  chunks.push({
    type: "finish" as const,
    finishReason: "stop" as const,
    usage: { promptTokens: 0, completionTokens: 0 },
  });
  return chunks;
}
```

This function converts a static response string into the LanguageModelV3StreamPart[] format that `simulateReadableStream` expects in the caching middleware.
