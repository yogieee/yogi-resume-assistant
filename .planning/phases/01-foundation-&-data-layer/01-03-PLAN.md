    The file must contain:
    - `@import "tailwindcss";` (keep existing)
    - `@import "tw-animate-css";` (keep if shadcn added it)
    - `@custom-variant dark (&:where(.dark, .dark *));` (keep if shadcn added it)
    - All shadcn/ui CSS variables (--background, --foreground, --card, etc.) -- KEEP these untouched
    - ADD console theme tokens inside `@theme`:
      - Glow accent colors: `--color-glow-cyan: oklch(0.82 0.15 192)`, `--color-glow-green: oklch(0.78 0.18 142)`, `--color-glow-purple: oklch(0.65 0.2 290)`, `--color-glow-amber: oklch(0.8 0.15 75)`, `--color-glow-red: oklch(0.65 0.2 25)`
      - Console background colors: `--color-console-bg: #0a0a0f`, `--color-console-surface: #12121a`, `--color-console-elevated: #1a1a2e`, `--color-console-border: #1e1e2e`
      - Console text colors: `--color-console-text: #e2e8f0`, `--color-console-text-dim: #64748b`, `--color-console-text-bright: #f8fafc`
      - Font families: `--font-sans: "Geist", ui-sans-serif, system-ui, sans-serif`, `--font-mono: "JetBrains Mono", ui-monospace, "Cascadia Code", "Fira Code", Menlo, Monaco, monospace`
      - Glow shadows: `--shadow-glow-sm`, `--shadow-glow-md`, `--shadow-glow-lg` (see research for values)
    - ADD in `@layer base`:
      - `html { color-scheme: dark; }`
      - `body { @apply bg-console-bg text-console-text font-mono antialiased; }`
      - Scrollbar styling for console aesthetic (webkit-scrollbar with console colors)

    **2. Update `src/app/layout.tsx`**

    Replace with font loading configuration:
    - Import `JetBrains_Mono` from `next/font/google` with `variable: "--font-mono"`, `subsets: ["latin"]`, `display: "swap"`
    - For Geist: Check what create-next-app generated. It may use `next/font/local` with bundled Geist files, or `next/font/google`. Keep whatever the scaffold generated for Geist/sans font, but ensure its variable is `--font-sans`. If no Geist font exists in scaffold, import `Geist` from `next/font/google` with `variable: "--font-sans"`.
    - Set metadata: title "Yogi Dev Console", description "AI-powered developer portfolio for Yogananda Govind"
    - Add `className={`dark ${jetbrainsMono.variable} ${geistFont.variable}`}` on `<html>` element
    - Add `lang="en"` on `<html>` element
    - Import `./globals.css`

    CRITICAL: The CSS variable names from next/font MUST match the @theme variable names. `--font-mono` in next/font maps to `--font-mono` in @theme. This is how Tailwind's `font-mono` utility picks up JetBrains Mono.
