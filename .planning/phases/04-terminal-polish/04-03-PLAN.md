  <name>Task 1: Create TypingEffect component</name>
  <files>src/components/terminal/typing-effect.tsx</files>
  <action>
  Create a new "use client" component at src/components/terminal/typing-effect.tsx.

  Props interface:
  - `children: string` -- the full text to reveal
  - `speed?: number` -- characters per second, default 40
  - `onComplete?: () => void` -- callback when animation finishes

  Implementation using motion primitives (NOT useState for character count -- avoid re-renders per character):
  - `const count = useMotionValue(0)` -- animated character count
  - `const rounded = useTransform(count, (v) => Math.round(v))` -- integer char count
  - `const [displayText, setDisplayText] = useState(children)` -- for the rendered text
  - `const [done, setDone] = useState(false)` -- animation complete flag

  Use a useEffect that:
  - Calculates duration as `children.length / speed`
  - Calls `animate(count, children.length, { type: "tween", duration, ease: "linear" })` to tween the count
  - On complete: set done=true, call onComplete
  - Returns controls.stop for cleanup

  Use another useEffect subscribing to `rounded.on("change", (latest) => setDisplayText(children.slice(0, latest)))` to update displayed text without triggering React re-renders per tick. Actually, since we need to render the sliced text, we do need a small state update. The key optimization is that motion handles the timing/animation frame loop, not setInterval.

  Alternative simpler approach that avoids the complexity: Use a single useState for characterCount and animate() to drive it. The animate() function from motion/react handles RAF timing efficiently:

  ```
  const [charCount, setCharCount] = useState(0)
  const [done, setDone] = useState(false)

  useEffect(() => {
    const controls = animate(0, children.length, {
      duration: children.length / speed,
      ease: "linear",
      onUpdate: (v) => setCharCount(Math.round(v)),
      onComplete: () => { setDone(true); onComplete?.() },
    })
    return () => controls.stop()
  }, [children, speed])
  ```

  Skip function: `count` to children.length immediately, set done=true, call onComplete.

  Render:
  - A span with onClick={skip} and className={done ? undefined : "cursor-pointer"}
  - Inside: `{children.slice(0, charCount)}`
  - If !done: append a blinking cursor span `<span className="animate-pulse text-glow-green">|</span>`

  IMPORTANT: All imports from "motion/react" (NOT "framer-motion"). Use `import { animate } from "motion/react"`.
  </action>
